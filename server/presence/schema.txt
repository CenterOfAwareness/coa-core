COA Presence JSON-DB Schema

{ coa_presence : {
    [system_name] : {   // Key must be a valid user alias on the local BBS
      [node_number] : { // Must be numeric, 0 through 255
        s : number,     // Node Status (nodedefs.js -> NodeStatus)
        a : number,     // Node Action (nodedefs.js -> NodeAction)
        u : string,     // User Alias
        c : string      // Custom status string
      }
    }
  }
}

Only a client authenticated as local user [system_name] can write to a location
under [system_name].  The entire DB is readable to all authenticated clients.
